# islettes-twirl

Personal/business website for Baton Twirling — team profiles, services, events, and online applications Baton Twirling Website A professional website for our baton twirling business that showcases the team, promotes classes and performances, lists upcoming events, and accepts online applications to join the team. Built as a lightweight, maintainable site to help clients find services and prospective performers apply easily.

Short site tagline / meta description (for header/meta) Baton twirling studio — meet the team, explore classes and events, and apply to join our performance team.
